Métodos

JavaScript - Métodos HTML DOM

¿Qué es?

Los métodos HTML DOM son acciones que puede realizar (en elementos HTML).
Las propiedades HTML DOM son valores (de elementos HTML) que puede establecer o cambiar.

Como funciona

Se accede a un elemento HTML comunmente mediante el id, y podemos manipular su informacion o su accion mediante un script

Uso

DOM permite acceder y manipular fácilmente etiquetas, ID, clases, atributos o elementos de HTML utilizando comandos o métodos proporcionados por el objeto del documento.

Componentes

JavaScript - Métodos HTML DOM

¿Cuales son?

Los objetos del DOM modelizan tanto la ventana del navegador como el historial, el documento o página web, y todos los elementos que pueda tener dentro la propia página, como párrafos, divisiones, tablas, formularios y sus campos, etc.

Estructura

La estructura del DOM está integrada por nodos, que se relacionan unos con otros en una estructura jerárquica, y cada nodo tiene un nodo superior del que depende, que es el nodo padre.

Uso

- Encontrar elementos HTML
-Cambio de elementos HTML
- Adición y eliminación de elementos
- Adición de controladores de eventos

JavaScript HTML DOM - Cambio de HTML

HTML DOM

El HTML DOM permite que JavaScript cambie el contenido de los elementos HTML.
La forma más sencilla de modificar el contenido de un elemento HTML es mediante la innerHTML propiedad.

Funciones

Cambios de contenido
Cambio de valor de un atributo
Contenido HTML dinamico

Uso

- Busqueda y modificacion
-Busqueda y eliminacion
- Busqueda y redireccion
- Busqueda mediante selectores

CSS

JavaScript HTML DOM - Cambio de CSS

CSS DOM

El HTML DOM permite que JavaScript cambie el estilo de los elementos HTML.

Uso de eventos

Los eventos son generados por el navegador cuando "le suceden cosas" a los elementos HTML:
- Se hace clic en un elemento
- La página ha cargado
- Los campos de entrada se cambian

Uso

- Cambios de color
- Cambio de animaciones al hacer click
- Cambios de fondos y formas
- Cambios de estilo totales
