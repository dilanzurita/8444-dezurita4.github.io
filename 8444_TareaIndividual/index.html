XML

JavaScript Navegación DOM HTML

¿Qué es?

Extensible Markup Language, que es un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos.

Estructura

Un archivo XML se divide en dos partes: prolog y body. La parte prolog consiste en metadatos administrativos, como declaración XML, instrucción de procesamiento opcional, declaración de tipo de documento y comentarios. La parte del body se compone de dos partes: estructural y de contenido (presente en los textos simples).

Uso

- El XML separa datos de HTML
- XML simplifica el intercambio de datos

- El XML aumenta la disponibilidad de datos

JSON

JavaScript HTML DOM Eventos

¿Qué es?

JSON, cuyas siglas significan en verdad JavaScript object notation es un formato de intercambio de datos que resulta muy fácil de leer y escribir para los programadores y sencillo de interpretar y crear para las máquinas.

Características

- Define seis tipos de valores: nulo, números, cadenas, booleanos, matrices y objetos
- Una coma o dos puntos mal ubicados pueden producir que un archivo JSON no funcione.
- Permite serializar y deserializar objetos en la mayoría de los idiomas.

Funcionamiento

Su funcionamiento se basa en la estructuración de una colección de pares con nombre y valor que contienen:
- Una clave: que corresponde al identificador del contenido
- Un valor: que representa el contenido correspondiente.

XML JSON
El formato XML soporta diferentes tipos de datos, entre ellos: imágenes y gráficos El formato JSON solo ofrece soporte a números y textos.
XML es más estructurado y, por lo tanto, tiene una interpretación más compleja. Los archivos JSON son fáciles de entender, pues su estructura y notación son bien sencillas.
Ambos soportan el formato UTF-8 el cual es el más utilizado en los sitios web
XML es legible por los humanos pero no necesariamente por las máquinas. JSON es ligero y fácil de leer, pero no proporciona información sobre el esquema o el tipo

AJAX

JavaScript HTML DOM EventListener

¿Que es?

AJAX significa JavaScript asíncrono
Es un conjunto de técnicas de desarrollo web que permiten que las aplicaciones web funcionen de forma asíncrona, procesando cualquier solicitud al servidor en segundo plano

Funcion

Uso

- Sistemas de votación y calificación
- Salas de chat
- Notificación de tendencias de Twitter
